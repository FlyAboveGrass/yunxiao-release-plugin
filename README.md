# Yunxiao Release Plugin

面向社区的 Codex / Claude Code 云效单 MR 发版插件。完整的安装、配置与使用指南见 [插件文档](plugins/yunxiao-release/README.md)。

在目标 Git 项目中选择对应宿主安装：

```bash
# Codex
curl -fsSL https://raw.githubusercontent.com/FlyAboveGrass/yunxiao-release-plugin/main/install.sh | bash

# Claude Code
curl -fsSL https://raw.githubusercontent.com/FlyAboveGrass/yunxiao-release-plugin/main/install-claude.sh | bash
```
